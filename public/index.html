Free chat

Using Node.js & jQuery


Key features

  • Updates 15.03.2018 - Add options to show last messages, 'loading' animation, render images
  • Updates 09.09.2018 - Add emoji support
  • Updates 05.03.2018 - Show who is typing
  • Updates 01.03.2018 - Added sound notifications
  • Updates 27.02.2018 - Using MongoDB to store messages, implemented first iteration for showing online users
  • Free chat rooms. You can access host/ChatName and get your own chat room which you can share with your friends.
  • Notifications on each sent message. When users sends messages, if you are online, you will be notified (if you allow this app to push notifications)
